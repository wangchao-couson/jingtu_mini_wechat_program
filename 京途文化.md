# 京途文化

> 和有趣的人，一起去探索更广阔的世界

## 一、京途是什么

京途不是旅行团。
不是简单的户外俱乐部。
也不是成长培训机构。

京途是一群有趣的人，一起探索世界、连接彼此、共同经历人生故事的青年探索型生活社区。

我们相信：
真正有价值的关系，不是在一次饭局中建立的。
而是在一次次共同经历中慢慢形成的。

一起走过山路。
一起骑过长坡。
一起看过日出。
一起经历风雨。

然后成为彼此人生中的老朋友。

## 二、京途的使命

让年轻人在城市中找到同频伙伴。
在探索中建立真实连接。
在长期同行中拥有更丰富的人生。

## 三、京途的愿景

成为北京最有温度、最有生命力、最值得长期参与的青年探索社区。

## 四、京途的定位

京途不是一个活动组织。
京途是一个探索型青年生活社区。

活动只是载体。
连接才是目的。
关系才是资产。
文化才是根基。

## 五、京途的核心逻辑

很多组织从成长开始。
京途从玩开始。

因为人与人的关系，不是靠说教建立的。
而是靠共同经历建立的。

京途相信：玩，是最好的连接方式。

发展路径：**玩 → 连接 → 信任 → 互助 → 成长**

| 阶段 | 关键词 | 说明 |
|------|--------|------|
| 第一阶段 | 玩 | 一起徒步、骑行、露营、旅行、探索城市 |
| 第二阶段 | 连接 | 认识更多有趣的人，发现更多不同的人生 |
| 第三阶段 | 信任 | 持续见面、持续同行，关系开始沉淀 |
| 第四阶段 | 互助 | 分享经验、分享资源、提供帮助、创造合作 |
| 第五阶段 | 成长 | 成长不是目标，是结果 |

## 六、京途相信什么

- **世界比想象中更大** — 人生不应该只剩下工位和出租屋
- **有趣比优秀更重要** — 有好奇心、有热情、有行动力、愿意尝试新事物的人
- **长期关系最有价值** — 不追求一次认识100个人，更希望十年后还能一起聊天、旅行、喝酒、合作
- **先创造价值** — 先给予，再获得。先帮助别人，机会自然会出现

## 七、京途文化关键词

| 关键词 | 含义 |
|--------|------|
| 探索 | 保持对世界的好奇心 |
| 真诚 | 真实做人，真诚相处 |
| 开放 | 尊重差异，拥抱不同 |
| 互助 | 让资源流动起来，让善意循环起来 |
| 有趣 | 保持热爱，保持生命力 |
| 长期主义 | 相信时间的力量，相信关系的复利 |

## 八、京途的同学文化

京途不倡导家文化。
因为家文化容易形成：辈分文化、权威文化、情绪绑架。

京途倡导**同学文化**。

大家因为兴趣相遇，因为探索同行，因为经历成长。

在京途没有大哥、没有家长、没有领导。
只有：新同学、老同学、同路人。

大家平等交流，共同建设社区，共同创造故事。

## 九、管理员的角色

管理员不是管理者，不是家长。

管理员是：
- **先行者** — 率先探索的人
- **组织者** — 把大家连接起来的人
- **引路人** — 帮助新人融入的人
- **文化守护者** — 维护社区氛围的人

## 十、活动原则

活动不是目的。关系才是目的。

活动设计优先级：
1. 互动感
2. 参与感
3. 体验感
4. 风景
5. 打卡

如果风景很好但没有交流、没有连接，这个活动并不成功。

## 十一、京途欢迎什么样的人

**欢迎：**
- 热爱生活的人
- 喜欢探索的人
- 愿意认识新朋友的人
- 愿意分享的人
- 愿意参与社区建设的人

**不欢迎：**
- 长期负能量的人
- 只索取不贡献的人
- 不尊重他人的人
- 破坏社区氛围的人

## 十二、京途的一句话

**主 Slogan：** 和有趣的人，一起去探索更广阔的世界。

**文化表达：** 先一起玩，再一起成为有故事的人。

**底层理念：** 因为探索而相遇，因为同行而成长。

## 十三、京途的终极目标

京途希望很多年以后，大家依然会记得：

曾经有一群有趣的人。
一起爬过山。
一起看过海。
一起走过很多路。
一起经历过人生的重要阶段。

然后成为彼此生命里长期存在的人。

这就是京途存在的意义。
