# 京途 · 年轻户外社区

> 北极星指引方向，京途陪伴成长

## 产品定位

京途 = **活动日历 + 照片墙 + 通讯录**

一群年轻人通过共同体验建立连接、获得归属感和长期成长的社区。  
所有活动交易通过微信群完成，小程序仅做展示与连接。

## 核心原则

- 能用微信群解决的问题，不做功能
- 能靠真实关系解决的问题，不做系统
- 优先创造连接 · 优先服务活动 · 优先记录故事

## 导航结构

| Tab | 页面 | 核心功能 |
|-----|------|---------|
| 🏠 | 今日京途 | 问候、近期活动（最多3个）、同学动态、精选照片、社区公告 |
| 🏔️ | 活动 | 招募中/即将开始/已结束 三Tab，活动详情 → 联系领队微信入群 |
| 👥 | 同学 | 搜索、8种兴趣标签筛选、同学档案、复制微信号 |
| 👤 | 我的 | 户外足迹统计、足迹地图、已加入活动、相册、故事 |

## 审核合规

| 项目 | 说明 |
|------|------|
| 主体类型 | 个人 |
| 类目建议 | 工具 → 信息查询 |
| 支付 | ❌ 无任何支付/价格 |
| 交易 | ❌ 无交易流程，引导至微信群 |
| 社交 | ❌ 无关注/私信，仅展示微信号供外部联系 |
| 内容 | ✅ 活动信息展示 + 照片 + 通讯录 |

## 运行方式

1. 微信开发者工具 → 导入项目
2. 填入 AppID → 编译预览
3. 初次启动自动初始化种子数据
