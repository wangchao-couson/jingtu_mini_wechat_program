# 京途 Human Interface Guidelines 1.0

> 京途不是一个让用户快速报名活动的小程序，而是一部不断生长的青年探索纪念册。

## 产品哲学

京途记录的不只是路线、照片和活动。

京途记录的是：

- 第一次一起看日出。
- 第一次一起爬上山顶。
- 第一次成为领队。
- 第一次提出一个被采纳的建议。
- 第一次认识一个后来成为朋友的人。

多年以后，大家再次打开京途，看到的不只是活动历史，而是那些共同经历过的故事，以及一路同行的人。

## 设计关键词

探索、温暖、留白、自然、故事、回忆、同路人。

京途不是社交软件，不是 OTA 旅行平台，也不是户外论坛。它更像苹果相册、国家地理和一本不断生长的青年同学录。

## 色彩

- 森林绿 `#2E5E4E`：探索、信任、自然。
- 米白 `#F7F5F1`：纸张、留白、记忆。
- 天空蓝 `#7FAFD4`：远方、开放、轻盈。
- 岩石灰 `#5F646D`：正文、说明、安静的信息。
- 落日橙 `#E58A4B`：只用于进行中、徽章、故事精选。

## 页面原则

- 顶部统一出现“京途”和一句每日文案。
- 页面不是工具入口，而是一段故事的开头。
- 不使用强促销、倒计时、报名转化式文案。
- 以“咨询领队”“了解行程”“一起走过的人”替代强转化和强社交文案。
- 照片必须真实、开阔、有人味，避免空泛素材感。

## 四个 Tab

- 故事：像翻相册，记录共同经历。
- 我们：一起走过的人，用关系和贡献表达连接。
- 发现：下一段故事从这里开始，包含队伍、路线灵感、精选路线、共创日志。
- 我的：个人成长、足迹、回忆和同行者。

## 动画节奏

动画要舒服，不炫技。

- 照片慢慢放大。
- 卡片轻轻浮起。
- 地图慢慢点亮。
- 头像轻轻弹开。
- 节点连线像树根，也像星座。

## 文案风格

短句，有留白，像旅行杂志的旁白。

推荐：

- 今天适合去看山。
- 下一次故事，也许就在周末开始。
- 有时候记住的不是风景，而是一起笑的人。

避免高压转化、价格促销、名额刺激和交易承诺。
