# 图片放置说明

后续新增实景图片可以放在这里。

推荐命名：

- `team-队伍名-cover.jpg`
- `team-队伍名-01.jpg`
- `role-姓名-avatar.jpg`
- `role-姓名-work-01.jpg`

在配置文件中引用时使用绝对路径，例如：

```js
cover: '/content/images/team-haituo-cover.jpg'
```

当前第一版主要使用本目录的 `magazine-*.jpg` 统一杂志风占位素材。

## 当前素材

`magazine-*.jpg` 为从 Unsplash 图片服务下载的真实户外/人物图片，用作 V1.0 体验版统一杂志风占位素材。后续正式上线前建议替换为京途真实活动素材或确认授权范围的图片。
